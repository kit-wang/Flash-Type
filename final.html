Hello! This is the main landing page for Flash Type. Our team, Kit Kat Juice, aims to create a Python program that functions as a typing speed test (with a GUI), which will test how fast the user can type a random paragraph (created through a program like generate_text and displayed one line at a time) based on characters/words per minute. At the start of each “round” of the typing test will have a mini-section that allows the user to choose the next text they will be tested on. At the end of the round, should the user choose to end it, error frequency analysis and the typing speed computations will be displayed (this will be made possible with the `time` module).

The Flash Type speed test uses a variety of texts taken from Project Gutenberg to allow for a more varied material to be tested on. Below are some examples of the texts used:

  • Metamorphosis
    • Written by Franz Kafka, "Metamorphosis" is about a salesman Gregor Samsa, who is suddenly transformed into a large insect.
  • Alice's Adventures in Wonderland
    • By Lewis Carroll, the story follows a young girl named Alice as she follows a white rabbit down a hole into a whimsical place.
  • Pride and Prejudice
    • "Pride and Prejudice" by Jane Austen follows the relationships between Elizabeth Bennet and Fitzwilliam Darcy, along with Jane Bennet and Charles Bingley. The four main characters' relationships with each other develop as they try to overcome their own feelings of pride and prejudice.
  • The Republic
    • "The Republic" discusses Plato's idea of justice, along with the idea of what a fair city-state government is.
  • Dubliners
    • "Dubliners" by James Joyce is a collection of short stories detailing various characters in the Irish city Dublin.
